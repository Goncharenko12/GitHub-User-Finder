## Описание
Программа позволяет искать пользователей GitHub по логину, просматривать информацию и добавлять их в избранное, сохраняющееся в JSON-файл.

## Как использовать API
- Введите логин пользователя GitHub в поле поиска.
- Нажмите кнопку "Найти" для поиска.
- Результаты отображаются в списке.
- Можно добавить пользователя в избранное.

## Примеры использования
- Ввод: `torvalds`, чтобы найти пользователя Линуса Торвальдса.
- Ввод: `octocat` для теста.

- библиотеки: requests, tkinter (шаблон уже встроен)
